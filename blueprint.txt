Phase 2: Data Preparation and Embedding Generation (10 Marks)

This directly addresses Question 1.

Step 2.1 Clean Text

Perform:

text = text.lower()

Remove:

[Applause]
[Interjections]
Page numbers
Extra spaces
Report Justification

Cleaning reduces noise and improves semantic representation quality.

Step 2.2 Tokenization

Use BERT tokenizer.

from transformers import AutoTokenizer

tokenizer = AutoTokenizer.from_pretrained(
    "bert-base-uncased"
)

Example:

tokenizer.tokenize(
    "education transformation policy"
)

Output:

['education',
 'transform',
 '##ation',
 'policy']
Why?

Assignment explicitly requests sub-word tokenization.

WordPiece satisfies this requirement.

Step 2.3 Generate Embeddings

Use Sentence-BERT.

from sentence_transformers import SentenceTransformer

model = SentenceTransformer(
    'all-MiniLM-L6-v2'
)

embeddings = model.encode(chunks)

Output:

(5000, 384)
Why?

Produces semantic embeddings suitable for:

Sentiment classification
Similarity search
FAISS
RAG

Step 4.1 Build FAISS Index
import faiss

index = faiss.IndexFlatL2(
    embeddings.shape[1]
)

index.add(embeddings)

Save:

faiss.write_index(
    index,
    "hansard.index"
)
Step 4.2 Retrieval

Example query:
